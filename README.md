# debianize-CMP

This repository contains the `debian/` from the actual source and the one generated by [debianize](https://salsa.debian.org/jelmer/debian-codemods/-/tree/master/debianize)

Notes: 
1. The results generated by `debianize` are not modified manually


## Usage

1. ./deb_diff 
2. ./deb_diff <number> (deb_diff 1)
